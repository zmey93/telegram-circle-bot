## ⚙️ Как работает

1. Пользователь отправляет видео боту (MP4, AVI, MOV и др.)
2. Бот скачивает видео через Telegram Bot API
3. `ffmpeg` обрезает видео до квадрата 1:1, масштабирует до 640×640
4. Битрейт **автоматически** рассчитывается под лимит 12 МБ Telegram
5. Готовый кружочек отправляется пользователю
6. Временные файлы автоматически удаляются

## 🚀 Быстрый старт

Для установки достаточно выполнить **одну команду** на сервере:

```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/zmey93/telegram-circle-bot/main/install.sh)"
```

Скрипт `install.sh` автоматически:

- ✅ Устанавливает Docker (если не установлен) и git
- ✅ Клонирует репозиторий в `/opt/circlebot`
- ✅ Запрашивает `BOT_TOKEN` от @BotFather
- ✅ Собирает Docker-образ и запускает контейнер
- ✅ Регистрирует команду `circlebot` в `/usr/local/bin`

> ⚠️ Требуется Ubuntu/Debian и права `root` (через `sudo`)

## 🔧 Установка вручную

Если хочешь всё сделать сам:

```bash
# 1. Клонируем репозиторий
git clone https://github.com/zmey93/telegram-circle-bot.git /opt/circlebot
cd /opt/circlebot

# 2. Создаём .env с токеном
echo "BOT_TOKEN=твой_токен" > .env

# 3. Собираем и запускаем
docker compose up -d --build

# 4. Проверяем логи
docker compose logs -f bot
```

## 📦 Управление ботом

После установки доступна команда `circlebot`:

| Команда | Действие |
|---------|----------|
| `circlebot start` | Запустить бота |
| `circlebot stop` | Остановить бота |
| `circlebot restart` | Перезапустить бота |
| `circlebot update` | `git pull` + пересборка + запуск |
| `circlebot status` | Статус Docker-контейнера |
| `circlebot logs` | Логи в реальном времени |
| `circlebot install` | Повторная установка |
| `circlebot uninstall` | Полное удаление бота |

## ⚠️ Ограничения

| Параметр | Значение | Причина |
|----------|----------|--------|
| Макс. длительность | **5 минут** | лимит бота |
| Макс. размер video note | **12 МБ** | лимит Telegram API |
| Макс. размер входного файла | **50 МБ** | лимит Bot API для скачивания |
| Размер вывода | **640×640 px** | требование Telegram |
| Битрейт | **авто** | под длину видео |

> 💡 Чем длиннее видео, тем ниже будет битрейт. Для 5-минутного видео битрейт составит ~́330 kbps.

## ❓ Частые вопросы

**Q: Бот не отвечает?**
A: Проверь логи: `circlebot logs`. Убедись, что в `.env` правильный `BOT_TOKEN`.

**Q: Как сменить токен?**
A: Отредактируй `/opt/circlebot/.env`, затем `circlebot restart`.

**Q: Почему кружочек квадратный, а не круглый?**
A: Кружочек Telegram отображает video note видео кругом самостоятельно. Файл сохраняется как MP4 640×640.

**Q: Можно отправь видео как файл?**
A: Нет. Telegram передаёт в Bot API качественно разные файлы. Отправляй видео как обычное видео.

**Q: Бот работает после перезагрузки сервера?**
A: Да, контейнер запущен с `restart: unless-stopped` — автостарт включён.
